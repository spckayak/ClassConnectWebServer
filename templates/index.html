ClassConnect is a cloud based application that integrates various technologies into a classroom setting. ClassConnect includes functionalities like a clicker or a poll based survey that are used for fast and dependable response systems. The difference from other clicker technologies is that ClassConnect will incorporate an analytics tool for real-time feedback of the information presented in a classroom.
